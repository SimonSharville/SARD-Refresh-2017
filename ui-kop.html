Rows & Perforations

Everything is contained within ROWs <div class="row"></div>. If you wish to add a perfration between 2 ROWs then simply add a new ROW between them with a class of sa-perf-grey <div class="row sa-perf"></div>.

The perforation is made of repeating dot images of the same colour as the body background colour. If you wish to have a different background colour than we will need to make new images of that colour and use them.

 

If you wish to have a coloured row inbetween 2 rows then do the following:
After the first row add a new row with an added class of .sa-perf, followed by a second row with two added classes of .row-[color] and .row-between-perfs.
This row is written:
<div class="row sa-perf"></div>
<div class="row row-grey row-between-perfs"></div>

This row is written:
<div class="row sa-perf"></div>
<div class="row row-turq row-between-perfs"></div>
