Style Guide

Welcome to the SARD Appraisal System Style Guide

Every effort has been made to be as clear as possible, but if you notice anything missing or have any questions, please contact me at design@simonsharville.co.uk

The system is built on the Bootstrap framework and wherever possible uses their standard components styled to fit the SARD identity. When this is not possible we have created a specific component, these have a prefix of sa- before the class i.e. a bootstrap Alert will have a class of class="alert alert-success" while a sard specific Alert will have a class of class="alert sa-alert-MRv".

The exception to this is Panels, due to their nested build of panel-heading, panel-body and panel-footer sitting inside a panel.

For this wireframe we haven't made all the navigation as live links, instead click on any link below to visit that page, to return to this page, just click on the the Header image.
